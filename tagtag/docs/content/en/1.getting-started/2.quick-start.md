# Quick Start

This guide will help you set up the Tagtag backend services and frontend application on your local machine.

## Prerequisites

Ensure you have the following installed:

### Backend
- **Java JDK 17+**
- **Maven 3.8+**
- **MySQL 8.0+**
- **Redis 6.0+**

### Frontend
- **Node.js >= 20.12.0**
- **pnpm >= 10.0.0**

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/admin/openwj/tagtag.git
cd tagtag
```

### 2. Initialize Database (Backend)

1.  Create a new MySQL database named `tagtag`.
2.  Run the SQL scripts located in `tagtag-test/resources/test-data.sql` (or the main initialization script if available).

### 3. Configure Application (Backend)

Edit `tagtag-start/src/main/resources/application.yml` to match your local environment (Database credentials, Redis host).

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/tagtag?useUnicode=true&characterEncoding=utf-8&useSSL=false&serverTimezone=Asia/Shanghai
    username: root
    password: your_password
  data:
    redis:
      host: localhost
      port: 6379
```

### 4. Build and Run Backend

```bash
mvn clean package -DskipTests
java -jar tagtag-start/target/tagtag-start-0.0.1-SNAPSHOT.jar
```

Alternatively, run `TagtagApplication.java` directly from your IDE.

### 5. Start Frontend Application

1.  Enter the frontend directory:
    ```bash
    cd tagtag-ui
    ```

2.  Install dependencies:
    ```bash
    pnpm install
    ```

3.  Start the development server:
    ```bash
    pnpm run dev
    ```

## Verification

Once started:

- **Frontend Application**: Access [http://localhost:5666](http://localhost:5666)
- **Backend API Docs**: Access [http://localhost:8080/doc.html](http://localhost:8080/doc.html) (Swagger/Knife4j)
