Tagtag is an enterprise-grade Java Spring Boot framework for the modern web. v1.0 includes all the core features needed to build robust admin systems.

### Core Features

- **Modular Architecture**: Strictly layered design ensuring decoupled and maintainable code.
- **Enterprise Ready**: Built-in RBAC, Data Permission, Operation Logs, and Rate Limiting.
- **Contract First**: Clear separation of API definitions and implementations using DTOs and Interfaces.
- **Modern Stack**: Built on Spring Boot 3, MyBatis Plus, and Redis.
