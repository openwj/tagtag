# 更新日志

## v1.0.0 (2025-12-12)

🎉 **Tagtag v1.0 正式发布！**

Tagtag 是一个面向现代 Web 的企业级 Java Spring Boot 框架。v1.0 版本包含了构建稳健后台系统所需的所有核心功能。

### 核心特性

- **模块化架构**: 采用 strictly layered design，确保代码解耦且易于维护。
- **企业级功能**: 内置 RBAC、数据权限、操作日志和接口限流。
- **契约优先**: 使用 DTO 和 Interface 清晰分离 API 定义与实现。
- **现代技术栈**: 基于 Spring Boot 3、MyBatis Plus 和 Redis 构建。
